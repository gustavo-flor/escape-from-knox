# Project Zomboid: Escape From Knox

A Zomboid mod based on Escape From Tarkov.

## Modules

- Health System
- First Aids
- Injectors and Stimulants

## Development

- **MacOS** / **Linux**

```sh
./install.sh <path-to-zomboid-mods-folder>
```

- **Windows**

```sh
install.bat <path-to-zomboid-mods-folder>
```
